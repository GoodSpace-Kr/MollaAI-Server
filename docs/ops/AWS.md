# AWS 운영 정보

이 문서는 MollaAI Server의 AWS 인스턴스와 네트워크 설정을 작업자가 빠르게 확인하기 위한 운영 메모이다. 실제 비밀값, 개인키, Access Key, DB 비밀번호는 이 문서에 기록하지 않는다.

## EC2 인스턴스

- Public IPv4: `43.202.22.150`
- 애플리케이션은 Docker/Nginx 구성으로 운영한다.
- 배포 절차와 Nginx 프록시 설정은 `docs/deploy/README.md`를 함께 확인한다.

## Security Group Inbound Rules

현재 인바운드 규칙은 아래와 같다.

| Port | Protocol | Source | 용도 |
| --- | --- | --- | --- |
| `443` | TCP | `0.0.0.0/0` | HTTPS |
| `3306` | TCP | `0.0.0.0/0` | MySQL |
| `22` | TCP | `0.0.0.0/0` | SSH |
| `80` | TCP | `0.0.0.0/0` | HTTP |

## 보안 주의사항

- `22`와 `3306`이 `0.0.0.0/0`에 열려 있으므로 운영 보안상 위험하다.
- 가능하면 SSH `22`는 관리자 고정 IP 또는 VPN 대역으로 제한한다.
- MySQL `3306`은 외부 전체 공개를 피하고, 애플리케이션 서버 또는 관리 대역에서만 접근하도록 제한한다.
- 운영 DB 비밀번호, SSH 개인키, AWS Access Key는 저장소에 커밋하지 않는다.
- 보안그룹을 변경한 경우 이 문서와 배포 문서를 함께 갱신한다.
